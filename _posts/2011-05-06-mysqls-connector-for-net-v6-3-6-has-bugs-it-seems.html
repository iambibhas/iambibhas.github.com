For some certain process, I needed to connect to the MySQL database from the .NET desktop app. So I searched a bit and saw that MySQL provides several connectors to connect to MySQL database using various languages or platforms, e.g. there are connectors for ODBC(forÂ Windows, Linux, Mac OS X, and Unix platforms), Java platform, .NET platform, MXJ(MBean for embedding the MySQL server in Java applications), C and C++ language and even for OpenOffice.org products! Seemed cool. went ahead and downloaded the latest stable version 6.3.6 of the .NET connector. But when I tried to install it, the installation failed unexpectedly. There were no error log for me to analyze and take step. Only this error message from the installer - [caption id="" align="aligncenter" width="510" caption="MySQL Connector/NET installation error"][/caption] Now I searched for the solution, went to the MySQL dev forum, but couldn't find a solution that worked. So I assumed that this installer is faulty, hence downloaded the immediate previous version 6.3.4 and tried to install it. Now it installed without any problem. Tried connecting from the .NET app, worked cool. So it's pretty much clear that the 6.3.6 installer has some glitch that needs to be worked on. Hope they fix it soon.
